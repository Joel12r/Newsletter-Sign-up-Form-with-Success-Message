# Newsletter Sign-up Form with Success Message

This project is a simple newsletter sign-up form with a success message, built using HTML, CSS, and JavaScript.

## Features

- Responsive design to ensure compatibility across various devices.
- Utilizes Bootstrap CSS framework for styling.
- Validates email input on submission.
- Displays a success message upon successful submission.
- Includes SVG icons for visual representation.

## Usage

To use this form, simply fill in your email address in the input field and click the "Subscribe to monthly newsletter" button.

## How to Run

1. Clone this repository.
2. Open the `index.html` file in your web browser.
3. Fill in your email address and submit the form.

## Technologies Used

- HTML
- CSS (Bootstrap)
- JavaScript
